# Online Ordering System with Blockchain Integration

## About The Project
This capstone thesis project is a secure and transparent online ordering platform that integrates blockchain technology to ensure data immutability and secure transaction logging. It was designed to address data integrity challenges in transactional systems, making order histories and payment records tamper-proof.

## Key Features
* Secure Transaction Logging: Utilizes blockchain concepts to create an immutable record of all system orders.
* Data Integrity & Protection: Protects against unauthorized data modification and ensures a highly transparent transaction history.
* Responsive User Interface: A clean, user-friendly frontend built for seamless customer interactions and efficient order management.

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone [https://github.com/yourusername/online-ordering-blockchain.git](https://github.com/yourusername/online-ordering-blockchain.git)
   ```

2. Navigate to the project directory:
   ```bash
   cd online-ordering-blockchain
   ```

3. Install frontend dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```
